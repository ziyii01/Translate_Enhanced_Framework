# Translate Enhanced Framework

## 食用方法

### 安裝依赖

```bash
pip install httpx
pip install loguru
pip install chardet
```

### 使用

```
::使用命令行键入
py main.py

::使用命令行传参
py main.py [command 1, command 2, ...]
```

## 鸣谢

本项目使用[繁化姬](https://docs.zhconvert.org/)

### 繁化姬注意事项

* 商業使用請參見[商業使用](https://docs.zhconvert.org/commercial/)。
